===========================================
SAU KHI UPLOAD XONG:
===========================================

1. SSH vào server:
   cd /www/wwwroot/afl-tracker/frontend

2. Cài đặt dependencies mới:
   npm install

3. Rebuild frontend:
   npm run build

4. Kiểm tra các file mới:
   ls -la src/config/
   ls -la src/hooks/
   ls -la src/pages/EnhancedDashboard.jsx
   ls -la src/pages/EnhancedCampaigns.jsx
